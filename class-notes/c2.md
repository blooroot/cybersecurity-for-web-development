# **Principios de seguridad: Las 3A**

- Autenticación: Alude a que el usuario pueda entrar a los sistemas con credenciales de acceso. 
- Autorización: Se refiere a qué cosa tiene acceso.
- Accountability: Auditoría. El sistema debe ser auditable, debemos saber qué ocurrió con nuestras instancias, usuarios, logs.

## **IAM: Identity and Access Management**
Servicio de AWS que ayuda con estas 3A: 
- A crear usuarios, asignarles roles asumidos por los mismos y políticas de acceso (autorización).
- Creación de contraseñas.
